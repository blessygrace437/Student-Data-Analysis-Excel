# Student Data Analysis using Excel

## Project Overview
Analyzed 500+ student records using Microsoft Excel
to identify performance trends and generate insights.

## Tools Used
- Microsoft Excel
- VLOOKUP, XLOOKUP
- Pivot Tables
- Power Query
- Charts & Dashboard

## Key Features
- Cleaned and organized 500+ student records
- Used VLOOKUP to search student by ID
- Created Pivot Table for grade distribution
- Built interactive performance dashboard

Data Cleaning | Data Visualization | Dashboard Development
